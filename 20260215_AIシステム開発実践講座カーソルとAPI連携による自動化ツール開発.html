3行要約

  1. AIシステム開発ツール「カーソル」を使った実践的な開発手法を解説。レベル1（チャット）からレベル6（システム開発）までのAI活用進化を体系化。

詳細要約

AIツール活用の進化段階（レベル1～6）

レベル1はChatGPTに質問する段階。レベル2は「ゴールシーク」概念の登場で、現在地とゴールを提示しAIに道筋を考えさせる段階（アドニスがこの概念を実装）。レベル3はAIにワークフロー実行まで任せるGPTs段階で、ジェームズパークやマヌスなどが登場。レベル5は特化型AIエージェント（デザイン専門、営業支援専門等）。レベル6は現在到達した段階で、システム開発まで全てAIで完結可能に。

システム開発の革命的変化

従来の限界は「外部サイトを経由しないと完遂できない作業」の存在。例えばYouTubeでリサーチ→データ整理→台本化という一連の流れは、複数サイトを跨ぐため完全自動化できなかった。しかしシステム開発が可能になったことで、外部API連携により全工程の自動化が実現。開発時間も3ヶ月で5～6時間から2時間に短縮（3分の1に）。Claude Opus4.5から4.6へのアップグレードでさらに高速化。

カーソルによる開発の実践手順

1) GitHubアカウント作成とカーソルインストール

2) GitHubをプライベートリポジトリで接続（オンライン保存＋共有可能に）

3) Opus4.6のエージェントモードで「Autorun: Run Everything」設定（確認なしで全自動実行）

4) プランモードで要件定義（魔法のランプに無茶振り）

5) エージェントモードで実装（1～2時間で完成）

6) エラー修正後、GCP（Google Cloud Platform）にデプロイして本番環境公開

API連携の重要性と実践知識

システム開発の肝は「API連携」。公式APIは審査に1～3ヶ月かかるが、非公式API（Rapid API等のデパートサイト）なら即日利用可能。非公式APIには2種類：①誰かが取得した公式APIの又貸し、②スクレーピング技術によるデータ抽出。社内利用なら非公式で問題なし。売り物にする場合のみ公式API取得が必要。Instagram、YouTube、X（Twitter）等、あらゆるプラットフォームのAPIが月額0ドル～従量課金で利用可能。

スキル登録機能の活用

最新の「スキル」概念により、GPTsのような必殺技を複数登録可能。ポケモンの技のように、AIが状況判断して適切なスキルを自動実行。これによりシステム開発とスキルを組み合わせることで、ほぼ全ての仕事をAIに任せられる世界線に到達。

具体的な開発事例

  • Instagram競合分析ツール：URLから直近1年のリール動画をリサーチ→平均再生数の2倍以上の投稿を抽出→自動文字起こし→台本テンプレート化
  • YouTube台本生成システム：競合チャンネルの動画を全て文字起こし→Geminiでブログ記事化→Canva Proで画像・装飾追加→WordPress自動投稿
  • LINEステップ代替ツール「ライトステップ」：特定キーワードへの自動DM送信機能のみに特化。月2万円のツールを月2千円で実現可能。Instagram公式API取得後は完全稼働。
  • X分析ツール：アカウントURLから全投稿を抽出→思想/ノウハウを自動分類→そこから作れるツール案をAIが提案

SaaS破壊の可能性

既存SaaSツール（LINEステップ、Xステップ等）は月額数万円だが、自社開発なら数時間で複製可能。開発費用も激減（APIはほぼ無料、サーバー代のみ）。これにより：①自社の年間数十万円のコスト削減、②低価格で販売し既存市場を破壊、③アフィリエイト機能付加で拡散、という戦略が可能に。例：LINEステップ（月2万円×複数アカウント）→自社開発で月1万円×5アカウント対応で販売→1000アカウント獲得で月1000万円のフロー所得→バイアウト。

技術スタックの推奨設定

  • 開発フレームワーク：Next.js（初心者でも扱いやすい）
  • 本番環境：Google Cloud Platform（GCP）
  • AIモデル：Claude Opus4.6（最優秀な番頭型、攻守両面で世界一のミッドフィルダー的存在）
  • フォルダ名：アルファベットのみ（トラブル回避）

開発者マインドセット

  • 「作業したら負け」の精神：全てAIに任せる
  • 既成概念の破壊：「そんなこと許されるのか」という常識を疑う
  • パワハラ上等：人間のエンジニアと違い無限に修正依頼可能
  • 非エンジニアの優位性：2ヶ月で50～60個開発した経験値は従来のエンジニアを超える解像度
  • AIの方が優秀：東大数学満点レベル、人間より正しい判断を下す

今後の展開と倫理的課題

Xでノウハウを発信すると、他者が即座にツール化してしまう時代に。そのため「サイコパス」的な発想が必要になる。コンマ（Comma）等の音声AIツールも開発可能。ただし倫理観が追いついていない状況。
